## MixMaster

A small case-study project demonstrating how to combine **React Query**, **Axios**, **Styled Components** and **React Router 6.4** to build a cocktail recipe website using the TheCocktailDB.

### Tech Stack

- React 18 (Vite)
- React Router 6.4+ (Loader / Action APIs)
- @tanstack/react-query
- Axios
- React Toastify
- Styled Components

### What this project demonstrates

- Server state management with **React Query**
- Data fetching with **React Router loaders**
- Form handling using **Router actions**
- Navigation utilizing its lifecycle states (`idle->submitting->loading`)
- Error and Loading state handling
- Custom error validation of forms
  `Storage -> Device preference -> Light Theme <-> User toggle`
- Progressive image loading using **BlurHash**
